## 🚀 Vercel Deployment:

1. **Vercel.com** pe jao
2. **Import GitHub repo**
3. **Environment Variables add karo:**
   ```
   LEAKOSINT_API_TOKEN = your_api_token
   ADMIN_PASSWORD = your_password
   ```
4. **Deploy!** ✅
